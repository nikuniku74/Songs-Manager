# Songs-Manager

## To install

1. `rm -rf venv`
2. `/Library/Frameworks/Python.framework/Versions/3.11/bin/python3 -m venv venv`
3. `source venv/bin/activate`
4. `pip install --upgrade pip setuptools wheel`
5. `pip install flask py2app`
6. `python setup.py py2app`
